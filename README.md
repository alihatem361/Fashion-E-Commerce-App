A modern fashion shopping app built with React Native & Expo. Beautiful pink/purple design with smooth onboarding and product browsing.

## Features

- 3-slide onboarding experience
- Product catalog with search and categories
- Product details with size/color selection
- Add to cart functionality
- Favorites and ratings

## Quick Start

```bash
# Install dependencies
npm install

# Start the app
npx expo start
```

Then scan the QR code with Expo Go app or press `w` for web browser.

## Tech Stack

- React Native & Expo
- TypeScript
- Expo Router (file-based navigation)
- AsyncStorage
- Ionicons
